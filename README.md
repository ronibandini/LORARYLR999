# Reyax Technologies LORA RYLR999
Sending and receiving data with Reyax Technologies LORA modules RYLR999 
I have included 2 examples (sender and receiver) for ESP32 using a Firebeetle board and 1 Python receiver example using Unihiker M10

# Hardware

1 x ESP32C6 Firebeetle 2

1 x Unihiker M10 

2 x LORA Reyax RYLYR999
